# Windaube Sound Tool
 Windows Sound Device Manipulation Tool written in Python

# dependencies 

- [pycaw](https://github.com/AndreMiras/pycaw)


- SoundVolumeView | Nirsoft (included)

[Official Website](https://www.nirsoft.net/utils/sound_volume_view.html)
